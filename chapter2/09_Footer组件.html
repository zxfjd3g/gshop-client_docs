# 9.3. 控制Footer界面的显示/隐藏

  • 解决方案:

    利用路由的meta配置和v-show

  • router/routes.js

{
  path: '/register',
  component: Register,
  meta: { // 需要隐藏footer的路由添加此配置
    isHideFooter: true
  }
},

{
  path: '/login',
  component: Login,
  meta: {
    isHideFooter: true
  }
},
  • App.vue
<Footer v-show="!$route.meta.isHideFooter"/>
